# Pink Ward

Pink Ward is an application for doing computer vision tasks and training models that requires 0 coding. It acts as an orchestration layer for inference, training, viewing metrics and file storage.

## Windows downloads

- `Pink-Ward-Installer-<version>-Windows-x64.zip`: per-user installer kit.
- `Pink-Ward-<version>-Windows-x64.7z`: portable build; extract it before use.
